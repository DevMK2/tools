# 1. compile_tar 
- tar 압축 해제 및 c 컴파일 후 실행 aka 채점
## dependency
1. python3
2. only run in bash shell terminal (나중에 zsh 추가 예정)

## usage
1. .bashrc 에 다음 입력
``` 
source {위치}/tools/setup.bash
```
2. 압축 파일이 모여있는 디렉토리로 이동해서 compile_tar 실행
3. 과제에서 요구한 문항 수 입력
4. 압축이 해제되면 아무키나 눌러서 컴파일 진행
5. 컴파일 다되면 아무키나 눌러서 프로그램 하나씩 실행
